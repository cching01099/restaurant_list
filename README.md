# Restaurant List

AlphaCamp 學期 2-3 W1 / A1:我的餐廳清單 專案

## 🎯專案功能

- 查看所有餐廳
- 點擊瀏覽餐廳的詳細資訊
- 點擊餐廳地址連到 Google Map
- 搜尋特定餐廳及特定美食類別

## ✔️開始使用

1. 請先確認有安裝 node.js 與 npm
2. 將專案 clone 到本地
3. 在本地開啟之後，透過終端機進入資料夾，輸入：

   ```bash
   npm install
   ```

4. 安裝完畢後，繼續輸入：

   ```bash
   npm run start
   ```

5. 若看見此行訊息則代表順利運行

   ```bash
   Express is listening on localhost:3000
   ```

6. 打開瀏覽器，並輸入 localhost:3000 進入首頁

## 📌開發工具

- Node.js v14.16.0
- Express 4.16.4
- Express-Handlebars @3.0.0
- Bootstrap v5.1
- Font-awesome V6.2.1
